간편결제 - 카카오페이

이전으로

  • PC

    카카오톡에 탑재되어 별도의 어플을 설치할 필요가 없습니다

    1. 1.

      결제수단 선택

      주문하기 결제수단을 간편결제 > 카카오페이 선택

      [결제하기] 클릭

    2. 2.

      휴대폰 번호/생년월일을 입력

      휴대폰 번호/생년월일 입력 후 [결제 요청] 클릭

    3. 3.

      카톡 메시지 확인

      [결제하기] 클릭하면 카카오 페이로 이동

    4. 4.

      카카오 페이지 카드 등록 및 선택

      카드 등록 및 선택 후 [결제하기] 클릭

    5. 5.

      카카오페이 비밀번호 또는 지문인증

      카카오페이 비밀번호 또는 지문 입력

    6. 6.

      카드 200만원 이상 결제 시 ARS 인증

      ARS 인증 요청 클릭 후 고객 인증(200만원 미만 결제 시 인증 페이지 미 노출)

    7. 7.

      결제 인증 완료

      [확인] 버튼 클릭

    8. 8. 결제완료
